# ブックマークレット設定ガイド（Android / iOS / PC）

> ⚠️ **このファイルに記載されているコードが唯一の公式ブックマークレット**（下記 deep-link 方式）。案内を求められたら **このコードをそのまま渡す**。clipboard コピー方式・共有シート方式など別物をその場で即興生成しない。

ブラウザで開いている記事の URL を数タップで Claude Code に送り、ブックマークとして整理する。
**外部アプリ不要・PAT 不要**。

> bootstrap 実行後はリポ名が自動置換済み。以下のコードをそのままコピーして使う。

---

## ブックマークレット（共通コード）

```
javascript:(function(){var r='kai-kou/claude-wiki-hub';location.href='https://claude.ai/code?repositories='+encodeURIComponent(r)+'&environment='+r.split('/')[1]+'&prompt='+encodeURIComponent('次のURLをブックマークして整理してにゃ：\n'+location.href);})();
```

> **bootstrap 未実行の場合**: `kai-kou/claude-wiki-hub` を `あなたのGitHubユーザー名/リポジトリ名` に手で置き換える（例: `my-user/my-wiki`）。

---

## Android（Chrome / Brave）

### 登録（一度だけ）

1. Chrome for Android で適当なページを開き **★** でブックマークする。
2. メニュー（⋮）→ **ブックマーク** → 今作ったブックマークを **長押し → 編集**。
3. **名前** を短くする（例: `bk`）。
4. **URL** を全部消し、上のブックマークレットを丸ごと貼り付けて **保存**。

### 使い方（3〜4 タップ）

1. ブックマークしたい記事を Chrome で開く。
2. **アドレスバーをタップ** し `bk`（付けた名前）と入力する。
3. 候補に出たブックマークレットをタップ → Claude Code が URL 入力済みで開く。
4. 内容を確認して **送信**（prefill の仕様で送信のみ手動）。

### 制約

- アドレスバーからの名前起動のみ動作する。ブックマーク一覧やホーム画面アイコンからのタップでは現在ページを認識できない（Chrome の仕様）。
- Firefox for Android は非対応。Chrome か Brave を使う。
- URL 保存時に先頭の `javascript:` が消えないよう、必ず **既存ブックマークの編集** から貼り付ける。
- どうしても動かない端末では、**記事の URL をコピーして Claude Code のチャットに直接貼る** だけでよい（`docs/rules/intent-routing.md` R-1 で同じくブックマーク化される）。

## iOS（Safari）

### 登録（一度だけ）

1. Safari で適当なページを開き、**共有ボタン（□↑）→ ブックマークに追加** をタップ。
2. **名前** を短くする（例: `bk`）。
3. 保存後、**ブックマーク（□に本のアイコン）→ 今作ったブックマークを長押し → 編集**。
4. **URL** を全部消し、上のブックマークレットを丸ごと貼り付けて **完了** をタップ。

### 使い方（3〜4 タップ）

1. ブックマークしたい記事を Safari で開く。
2. **アドレスバーをタップ** し `bk`（付けた名前）と入力する。
3. 候補に出たブックマークレットをタップ → Claude Code が URL 入力済みで開く。
4. 内容を確認して **送信**。

### 制約

- アドレスバーからの候補起動のみ動作する（ホーム画面追加・ブックマーク一覧タップは現在ページを認識できない）。
- Firefox for iOS は非対応。Safari を使う。

## PC（Chrome / Firefox / Safari）

### Chrome

**登録:**

1. ブックマークバーを **右クリック → ページを追加**。
2. **名前** を短くする（例: `bk`）。
3. **URL** フィールドに上のブックマークレットを貼り付けて **保存**。

> ブックマークバーが非表示の場合: `Ctrl+Shift+B`（Mac: `⌘+Shift+B`）で表示する。

**使い方:**

1. ブックマークしたい記事を Chrome で開く。
2. ブックマークバーの **`bk`** をクリック → Claude Code が URL 入力済みで開く。
3. 内容を確認して **送信**。

---

### Firefox

**登録:**

1. メニュー（≡）→ **ブックマーク → ブックマークを管理**。
2. ツールバーの **新しいブックマーク（☆+）** をクリック。
3. **名前** を短くする（例: `bk`）。
4. **URL** フィールドに上のブックマークレットを貼り付けて **追加**。

> ブックマークツールバーが非表示の場合: `表示 → ツールバー → ブックマークツールバー → 常に表示` で有効化。

**使い方:**

1. ブックマークしたい記事を Firefox で開く。
2. ブックマークツールバーの **`bk`** をクリック → Claude Code が URL 入力済みで開く。
3. 内容を確認して **送信**。

---

### Safari（Mac）

**登録:**

1. メニュー → **ブックマーク → ブックマークを追加**。
2. **追加先: お気に入りバー** を選び **追加**。
3. お気に入りバーに追加されたブックマークを **右クリック → 編集アドレス**。
4. URL を全部消し、上のブックマークレットを貼り付けて **Return**。
5. 名前も右クリックで短く変更（例: `bk`）。

> お気に入りバーが非表示の場合: `表示 → お気に入りバーを表示` で有効化。

**使い方:**

1. ブックマークしたい記事を Safari で開く。
2. お気に入りバーの **`bk`** をクリック → Claude Code が URL 入力済みで開く。
3. 内容を確認して **送信**。

---

## 動作の仕組み

ブックマークレットをクリック / タップすると、以下の URL に遷移する:

```
https://claude.ai/code
  ?repositories=<あなたのリポジトリ>
  &environment=<リポジトリ名>
  &prompt=次のURLをブックマークして整理してにゃ：<記事のURL>
```

Claude Code が起動し、`intent-routing.md` の **R-1（bookmark）** ルートで処理される:
タイトル取得 → `bookmarks/inbox.md` に追記 → 自動タグ付け → インデックス更新。
